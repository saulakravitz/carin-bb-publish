ValueSet: NUBC Dispensed As Written Or Product Selection Code

Summary

Defining URL:http://hl7.org/fhir/us/ValueSet/ncpdp-dispensedaswrittenorproductselectioncode
Version:0.1.0
Name:NUBCDispensedAsWrittenOrProductSelectionCode
Status:draft
Title:NUBC Dispensed As Written Or Product Selection Code
Definition:

Prescriber's instruction regarding substitution of generic equivalents or order to dispense the specific prescribed medication. NCPDP field # 408-D8 (Dispensed As Written (DAW)/Product Selection Code).

Publisher:HL7 Financial Management Working Group
Source Resource:XML / JSON / Turtle
