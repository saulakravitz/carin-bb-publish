ValueSet: AMA CPT CMS HCPCS Procedure Codes

Summary

Defining URL:http://hl7.org/fhir/us/ValueSet/ama-cpt-cms-hcpcs-procedure-codes
Version:0.1.0
Name:AMACPTCMSHCPCSProcedureCodes
Status:draft
Title:AMA CPT CMS HCPCS Procedure Codes
Definition:

CPT (HCPCS I) and HCPCS II Procedure Codes.

Publisher:HL7 Financial Management Working Group
Source Resource:XML / JSON / Turtle

References

This value set is not used here; it may be used elsewhere (e.g. specifications and/or implementations that use this content)

Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Include all codes defined in http://www.ama-assn.org/go/cpt, where the codes are contained in http://hl7.org/fhir/ValueSet/cpt-all
  • Include all codes defined in urn:oid:2.16.840.1.113883.6.285

 

Expansion

No Expansion for this valueset (not supported by Publication Tooling)
