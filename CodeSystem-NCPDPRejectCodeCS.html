CodeSystem: NCPDP Reject Code

Summary

Defining URL:http://ncpdp.org/reject-code
Version:0.4.1
Name:NCPDPRejectCodeCS
Title:NCPDP Reject Code
Status:Active
Content:None of the concepts defined by the code system are included in the code system resource
Definition:

The National Council for Prescription Drug Programs, (NCPDP) Reject Code (511-FB) code set contains exception definitions for use when transaction processing cannot be completed.

Publisher:HL7 Financial Management Working Group
Content Mode:Not Present

Notes:

This stub CodeSystem resource represents the proprietary National Program for Prescription Drug Programs (NCPDP) Reject Code (511-FB) code set–enabling it to be referenced within this guide.
